To install package needed:

1. open cmd or Terminal
2. run 'pip instal django-bootstrap3'
3. run 'pip instal django-bootstrap3-datetimepicker-3'
4. run 'pip instal django-phonenumber-field'
5. run 'pip instal pandas'
